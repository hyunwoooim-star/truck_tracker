## 시작하기

### 회원가입

1. 앱 실행
2. **"회원가입"** 버튼 클릭
3. 정보 입력:
   - 이메일 주소
   - 비밀번호 (최소 8자, 대문자+소문자+숫자+특수문자 필요)
4. 약관 동의 체크
5. **"가입하기"** 버튼 클릭

### 로그인

1. 이메일과 비밀번호 입력
2. **"로그인"** 버튼 클릭
3. (선택) "둘러보기"로 게스트 모드 사용 가능

## 고객용 기능

### 1. 트럭 찾기

#### 지도에서 찾기
1. 하단 탭에서 **"지도"** 선택
2. 지도에 표시된 마커 확인
   - 🟢 녹색: 영업 중
   - 🔴 빨간색: 영업 종료
3. 마커 탭하여 트럭 정보 미리보기
4. 상세 정보 보려면 마커 클릭

#### 목록에서 찾기
1. 하단 탭에서 **"목록"** 선택
2. 필터 사용:
   - 음식 종류 (한식, 중식, 양식 등)
   - 영업 상태 (영업 중만 보기)
   - 거리 순 정렬
3. 검색창에 트럭명 또는 음식 입력

### 2. 트럭 상세 정보

#### 메뉴 보기
- 스크롤하여 전체 메뉴 확인
- 가격, 설명 포함

#### 영업 일정 확인
- 요일별 영업 시간 표시
- 현재 영업 중인지 실시간 확인

#### 위치 확인
- 지도에서 현재 위치 표시
- 내 위치에서 거리 계산

### 3. 리뷰 작성

1. 트럭 상세 페이지에서 **"리뷰 작성"** 버튼 클릭
2. 별점 선택 (1-5점)
3. 리뷰 내용 입력 (선택)
4. 사진 첨부 (선택):
   - **"사진 추가"** 버튼
   - 갤러리 또는 카메라 선택
5. **"등록"** 버튼 클릭

### 4. 즐겨찾기

#### 추가하기
- 트럭 상세 페이지에서 ⭐ 아이콘 탭

#### 관리하기
- 하단 탭 **"즐겨찾기"** 에서 확인
- 즐겨찾기한 트럭 목록 보기
- 다시 탭하여 제거

### 5. 주문하기

1. 트럭 상세 페이지에서 **"주문하기"** 버튼
2. 메뉴 선택 및 수량 입력
3. 장바구니 확인
4. **"주문 완료"** 버튼
5. 주문 상태 실시간 확인:
   - ⏳ 접수 대기
   - 🔥 준비 중
   - ✅ 완료

### 6. QR 체크인

1. 트럭 방문 시 QR 코드 스캔
2. 자동으로 방문 기록 저장
3. 로열티 포인트 적립

## 사장님용 기능

### 1. 대시보드 접근

- 로그인 후 자동으로 사장님 대시보드로 이동
- 실시간 통계 확인:
  - 오늘 매출
  - 주문 수
  - 리뷰 수

### 2. 영업 상태 관리

#### 영업 시작/종료
1. 대시보드 상단 토글 스위치
2. **"영업 시작"** / **"영업 종료"** 선택
3. 고객 지도에 실시간 반영

#### 위치 업데이트
1. **"위치 업데이트"** 버튼 클릭
2. 지도에서 현재 위치 선택
3. **"위치 저장"** 버튼
4. 고객 앱에 즉시 표시

### 3. 주문 관리

#### Kanban 보드
- 드래그 앤 드롭으로 상태 변경:
  - **접수 대기** → **준비 중** → **완료**
- 주문 상세 정보:
  - 고객명
  - 주문 시간
  - 메뉴 및 수량

#### 주문 알림
- 새 주문 시 푸시 알림 수신
- 알림 탭하여 바로 주문 확인

### 4. 메뉴 관리

1. 대시보드 → **"메뉴 관리"** 메뉴
2. 새 메뉴 추가:
   - 메뉴명, 가격, 설명 입력
   - 사진 업로드
3. 기존 메뉴 수정/삭제
4. 품절 표시 토글

### 5. 통계 및 분석

#### 매출 통계
- 일별, 주별, 월별 매출 그래프
- 전월 대비 증감률

#### 인기 메뉴
- 판매량 순위
- 매출 기여도

#### 고객 분석
- 신규 vs 재방문 고객
- 리뷰 평균 별점

### 6. 공지사항 작성

1. 대시보드 → **"공지사항"** 버튼
2. 제목과 내용 입력
3. **"발송"** 버튼
4. 팔로워에게 푸시 알림 전송

### 7. 쿠폰 발행

1. **"쿠폰 생성"** 메뉴
2. 쿠폰 정보 입력:
   - 할인율 또는 할인 금액
   - 유효 기간
   - 최대 사용 횟수
3. QR 코드 자동 생성
4. 고객이 스캔하여 사용

## FAQ

### Q: 비밀번호를 잊어버렸어요
**A**: 로그인 화면 → **"비밀번호 찾기"** → 이메일 입력 → 재설정 링크 수신

### Q: 리뷰를 수정하고 싶어요
**A**: 현재 리뷰 수정 기능은 지원하지 않습니다. 리뷰 삭제 후 재작성 가능합니다.

### Q: 주문을 취소하고 싶어요
**A**: 주문 상세 페이지 → **"취소 요청"** 버튼 → 사장님 승인 필요

### Q: 영업 시작 알림을 받고 싶어요
**A**: 트럭 상세 페이지 → ⭐ 즐겨찾기 추가 → 알림 자동 수신

### Q: GPS 위치가 정확하지 않아요
**A**:
1. 설정 → 위치 권한 확인
2. GPS 정확도 "높음"으로 설정
3. 실외에서 앱 재시작

### Q: 푸시 알림이 안 와요
**A**:
1. 설정 → 앱 알림 권한 확인
2. 배터리 절약 모드 해제
3. 앱 강제 종료 후 재실행

### Q: 사장님 모드로 전환하려면?
**A**:
1. 로그아웃
2. 사장님 계정으로 로그인
3. 자동으로 대시보드 표시

### Q: 여러 트럭을 운영하는데 어떻게 관리하나요?
**A**: 현재는 1개 계정당 1개 트럭만 지원합니다. 추후 업데이트 예정입니다.

### Q: 오프라인에서도 사용할 수 있나요?
**A**: 실시간 데이터 동기화가 필요하여 인터넷 연결이 필수입니다. (오프라인 모드 개발 예정)

## 문제 해결

### 앱이 느려요
1. 앱 캐시 삭제: 설정 → 앱 → 저장공간 → 캐시 삭제
2. 최신 버전 업데이트 확인
3. 기기 재시작

### 지도가 로딩되지 않아요
1. 인터넷 연결 확인
2. 위치 권한 허용 확인
3. 앱 재시작

### 로그인이 안 돼요
1. 이메일/비밀번호 확인
2. 대소문자 구분 주의
3. 비밀번호 찾기 이용

### 사진 업로드가 실패해요
1. 저장공간 확인 (최소 100MB 필요)
2. 사진 권한 허용 확인
3. 사진 크기 확인 (최대 5MB)
4. Wi-Fi 연결 시 재시도

### 결제가 안 돼요
**참고**: 현재 버전은 현장 결제만 지원합니다. 온라인 결제는 추후 업데이트 예정입니다.
